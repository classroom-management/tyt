Instruction

Use your imagination and creativity to create 3D objects with the game build tools

Build block

Click the hammber icon to build a new block

Destroy block

Click the trash icon to destroy a block

Scaling

Use the scale up and scale down icon to change current block size

Painting

Use the paint tool to paint current blocks with new pattern

Block Type

You can choose between block type and plane type

Color and pattern

Pick many color and pattern to decorate the scene
